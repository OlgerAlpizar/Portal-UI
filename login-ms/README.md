# Fekolti Users Micro-Service

- Node
- Express
- Mongo
- TypeScript

## What is implemented?

- Helmet provide more security on HTTP requests
- Mongo Atlas for Mongo instance
- Mongoose ODM object document mapper
- Global error hanlder
- DTO's
- Class validations
- Swagger
- Morgan and winston
- File upload
- Prettier and EsLint for linter and format
